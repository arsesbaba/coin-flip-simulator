# coin-flip-simulator
A Python simulation demonstrating probability theory and the Law of Large Numbers.

## What it does
Simulates thousands of coin flips and visualizes how the probability of heads converges to 50% as the number of flips increases.

## Why I built this
As an aspiring quantitative finance professional, understanding probability at a deep level is foundational. This project is my first step toward building more complex financial simulations.

## What I learned
- How randomness behaves over large sample sizes
- The Law of Large Numbers in action
- Python fundamentals and data visualization

## Technologies
- Python
- NumPy
- Matplotlib
